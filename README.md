# Hide Chats for WhatsApp Web

A lightweight browser extension that lets you hide chats on **WhatsApp Web** to improve privacy and reduce distractions.

🌐 Firefox Add-on: https://addons.mozilla.org/en-US/firefox/addon/hide-chats-for-whatsapp-web/

> ⚠️ This project is **NOT affiliated with WhatsApp or Meta**.

---

## ✨ Why This Extension Exists

WhatsApp Web does not provide a native way to hide chats or protect conversations from people nearby.

This extension was built to solve a simple problem:

* 👀 Prevent shoulder-surfing in public spaces
* 🧠 Reduce visual distraction
* 🔒 Improve privacy while working or studying

Many users rely on browser extensions for privacy improvements because WhatsApp Web lacks built-in controls for interface visibility.

---

## 🚀 Features

* Hide selected chats instantly
* Clean and minimal UI integration
* Works directly inside WhatsApp Web
* No login required
* No analytics or tracking
* Lightweight and fast

---

## 🔒 Privacy First

This extension:

* ✅ Does **not** collect user data
* ✅ Does **not** send information to external servers
* ✅ Runs entirely inside the browser
* ✅ Only accesses `web.whatsapp.com`

Your messages never leave your device.

---

## 🧩 How It Works

1. Install the extension
2. Open WhatsApp Web
3. Reload the page
4. Hide chats using the extension controls

---

## 🛠 Development

Clone the repository:

```bash
git clone https://github.com/yunusayan0/hide-chats-whatsapp-web.git
```

Load temporarily in Firefox:

1. Open `about:debugging`
2. Select **This Firefox**
3. Click **Load Temporary Add-on**
4. Choose `manifest.json`
